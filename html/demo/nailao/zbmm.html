私募证券投资基金经理

任职要求：

1. 5年以上证券投资领域工作2经验；

2. 熟悉股票、股指期货、债券、金融衍生品等二级市场基金产品的设立、幕资、投资、风控和投后工作，具备国内外知名投资银行与资产管理公司工作经验优先；

3. 有扎实的财务、金融知识，有良好的投融资分析、判断能力及决策力，熟悉项目的尽职调查、可行性研究分析、项目管理等；

4. 会运用公司自由现金流估值模型、股息贴现估值模型等方法进行估值；

5. 善于学习，具有较强的信息分析能力、创新和组织协调能力、团队合作和敬业精神，责任心强，具有良好的道德品质和职业操守。

投资研究经理

任职要求：

1. 有国内阳光私募基金工作经验，且3年以上实盘记录；

2. 具备从事投资活动所需要的金融行业知识和技能，知识广博，有敏锐的投资洞察力，熟悉行业分析；

3. 证券实盘操盘功底扎实,有丰富实战经验,具有良好的职业道德和风险防范意识,能提供1年以上持续历史操作记录 ;

4. 工作有条理,主动性强,擅于沟通,有较强的独立研究学习能力;

5. 工作细心谨慎,责任心强，诚实守信可承。

风险控制经理

任职要求：

1. 金融、法律、财务等相关专业本科及以上学历, 具有基金从业资格;

2. 6年以上金融行业法律合规、风险控制或专职律师工作经验，曾任职于大型金融机构、曾参与IPO或新三板挂牌项目的运作、或具有律师资格证者有先；

3. 精通基金行业股权投资工作流程，熟悉公司法、合同法、担保法、破产法、合作企业法等领域法律理论与实务；

4. 诚实守信，为人正直，具有较强的沟通协调能力、执行能力、团队合作能力；

5. 具有较强的分析、写作能力，具备编写业务风险监控报告及法律尽职调查报告的能力。
